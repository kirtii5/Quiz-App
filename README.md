# 📘 Quizly 

**Quizly** is a full-stack quiz application built to simplify assessments for students and educators. It provides a smooth and intuitive interface to create, share, and take quizzes with ease.

---

## 📄 Project Description

Quizly enables users to either **create** or **attempt quizzes** using unique quiz codes. Designed for both students and teachers, this app supports:
- Manual quiz creation
- Excel-based quiz uploads
- Real-time quiz taking
- Automatic result calculation and storage

---

## 🛠 Tech Stack

**Frontend**:  
- React (Vite)  
- JavaScript (ES6+)  
- Tailwind CSS  

**Backend**:  
- Node.js  
- Express.js  
- MongoDB (via mongo-atlas)  
- JWT for authentication  
- RESTful APIs  

---

## ✨ Features

- 🔐 User Authentication (JWT-based)
- 📤 Upload questions via Excel or manual form
- 🧪 Take quizzes with timer and auto-submit
- 📊 View results (user + admin side)
- 🎯 Evaluate multiple question types (MCQ single, multiple, write)
- 🎨 Clean, responsive UI
- 🌐 Deployed frontend at: [quizlyy.netlify.app](https://quizlyy.netlify.app)

## 🧩 Installation / Setup Instructions

### Clone the repository
```bash
git clone https://github.com/kirtii5/Quiz-App.git
```
### 📦 Backend Setup
```bash
cd Quiz-App/backend
npm install
```
### Frontend Setup
```bash
cd ../frontend/Quiz-app
npm install
npm run dev
```
Frontend runs at: http://localhost:5173
